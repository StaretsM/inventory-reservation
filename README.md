# Reservation Service

---
## Запуск приложения через Docker
### 1. Сборка проекта
Из корневой директории проекта выполнить:
```
mvn clean package
```
После сборки будет создан jar‑файл приложения в папке `target`.

### 2. Запуск контейнеров
В корневой директории проекта выполнить:
```
docker compose up --build
```
Будут запущены:
* PostgreSQL
* Приложение reservation-service

---

## Остановка приложения
Чтобы остановить контейнеры, выполнить:
```
docker compose down
```
Если нужно удалить все данные из базы:
```
docker compose down -v
```

## Запуск тестов
Тесты запускаются стандартной командой Maven.
```
mvn test
```
